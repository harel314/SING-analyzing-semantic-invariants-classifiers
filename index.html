Abstract

SING analyzes semantic structure in classifier representations via null-space geometry. The method projects features to remove null components, translates them into a shared embedding space, and quantifies semantic changes with image-level and attribute-level scores.

This repository provides a lightweight, reproducible implementation for single-image analysis, including model loading, translator loading, principal projection, metric computation, and generation.

Key Contributions

  • Null-space based semantic analysis pipeline for pretrained classifiers.
  • Model-agnostic translator workflow with metadata and checkpoint validation.
  • Single-image metrics and generation workflow for reproducible analysis.
  • Ready-to-run notebook and CLI paths for practical usage.
