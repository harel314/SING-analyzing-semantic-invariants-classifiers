SING: Analyzing Semantic Invariants in Classifiers

CVPR 2026 (Accepted)

Official project page for the SING paper.

Abstract

All classifiers, including state-of-the-art vision models, contain invariants induced by the geometry of their linear mappings. These invariants reside in the classifier null space and produce equivalent inputs with identical outputs.

SING (Semantic Interpretation of the Null-space Geometry) translates this hidden geometric behavior into human-interpretable semantics by mapping classifier features into a multimodal space and quantifying semantic drift. The framework supports single-image analysis and model-level comparisons.

Method Overview

SING decomposes classifier-head weights with SVD to obtain principal/null projectors, maps features to a vision-language space via a translator, constructs equivalent pairs through null-space operations, and measures semantic effects with textual and visual diagnostics.

Key Contributions

  • Semantic interpretation of null-space invariants using equivalent-pair analysis.
  • Cross-architecture protocol to compare semantic leakage into invariant directions.
  • Open-vocabulary class analysis for detecting spurious semantic correlations.
  • Practical single-image workflow with reproducible metrics and generated examples.
